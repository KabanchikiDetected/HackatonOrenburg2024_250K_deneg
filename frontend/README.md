`libs`: @vkruglikov/react-telegram-web-app


## Docs

[docs](docs/index.md)

## Run

1. clone this repo
2. Navigate to the directory `/src/frontend`:  
`cd src/frontend`
3. a) for dev:  
`npm run dev`  
b) for build:  
`npm run build`
